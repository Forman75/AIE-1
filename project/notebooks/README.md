# Ноутбуки проекта

Экспериментальные ноутбуки. Основной код вынесен в `src/` — ноутбуки лишь импортируют его и визуализируют результаты.

- `01_eda.ipynb` — разведочный анализ данных: распределения, пропуски, дисбаланс классов, связь признаков с оттоком.
- `02_baselines.ipynb` — базовая модель (LogisticRegression) и сравнение с ансамблями по кросс-валидации.
- `03_models_tuning.ipynb` — эксперименты с гиперпараметрами, анализ значимости признаков, выбор финальной модели.

## Запуск

```bash
cd project
pip install -r requirements.txt
jupyter notebook
```

Ноутбуки рассчитаны на запуск из каталога `project/` (импорты вида `from src...`).
