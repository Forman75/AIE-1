# Данные проекта

## Источник

Проект использует **синтетические данные** — реальные клиентские данные содержат персональную информацию и не могут использоваться (см. требования курса).

Датасет генерируется программно модулем `src/data/generate.py`. Его структура повторяет открытый датасет *Telco Customer Churn*, а целевая переменная `churn` формируется по осмысленной логит-модели с нелинейными эффектами и взаимодействиями признаков. Генерация детерминирована при фиксированном `seed`.

## Файлы

- `sample_customers.csv` — демонстрационная выборка (200 клиентов), стратифицированная по `churn`. Создаётся автоматически при `python -m src.train`. Используется для быстрых проверок и примеров запросов к сервису.

Полный датасет (7000 строк) **не хранится в репозитории** — он детерминированно воспроизводится при обучении. Это соответствует требованию не коммитить крупные выгрузки.

## Схема признаков

| Признак | Тип | Описание |
|---|---|---|
| `customer_id` | str | идентификатор клиента |
| `senior_citizen` | 0/1 | пожилой клиент |
| `partner` | Yes/No | наличие супруга(и) |
| `dependents` | Yes/No | наличие иждивенцев |
| `tenure` | int | срок обслуживания, мес. (0–72) |
| `contract` | cat | тип контракта (Month-to-month / One year / Two year) |
| `internet_service` | cat | DSL / Fiber optic / No |
| `online_security` | cat | Yes / No / No internet service |
| `tech_support` | cat | Yes / No / No internet service |
| `streaming_tv` | cat | Yes / No / No internet service |
| `monthly_charges` | float | ежемесячные траты |
| `total_charges` | float | суммарные траты (есть пропуски) |
| `payment_method` | cat | способ оплаты |
| `paperless_billing` | Yes/No | безбумажный биллинг |
| `num_support_calls` | int | число обращений в поддержку |
| `churn` | 0/1 | **целевая переменная**: ушёл ли клиент |
