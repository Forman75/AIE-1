# Конфигурационные файлы

Все настраиваемые параметры проекта вынесены сюда — менять поведение можно без правки кода.

- `config.yaml` — общие настройки: параметры датасета, пути к артефактам.
- `training.yaml` — реестр моделей, гиперпараметры, схема кросс-валидации, метрика выбора финальной модели.
- `inference.yaml` — параметры инференса: пороги риск-категорий, число факторов.
- `.env.example` — **шаблон** переменных окружения (без значений секретов).

## Работа с секретами

- Реальный файл `.env` создаётся локально из `.env.example` и **не коммитится** (он в `.gitignore`).
- В репозитории — только `.env.example` с именами переменных.
- Подробнее см. `SECURITY.md` в корне проекта.
