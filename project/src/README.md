# Исходный код проекта

Основной код проекта. Структура по слоям:

- `data/` — генерация и загрузка данных:
  - `generate.py` — синтетический генератор датасета оттока;
  - `loader.py` — загрузка, схема признаков, разбиение train/test.
- `features/` — препроцессинг:
  - `preprocess.py` — `ColumnTransformer` (импьютация, масштабирование, one-hot).
- `models/` — обучение и эксперименты:
  - `registry.py` — создание моделей по конфигу;
  - `trainer.py` — кросс-валидация, метрики, сравнение, выбор модели;
  - `tracking.py` — лёгкий трекинг экспериментов (JSON);
  - `plots.py` — графики (ROC, матрица ошибок, значимость признаков).
- `service/` — REST API (запускаемый пакет):
  - `app.py` — фабрика Flask-приложения и эндпоинты;
  - `wsgi.py` — WSGI-объект для production-сервера (waitress);
  - `__main__.py` — точка входа `python -m src.service`;
  - `schemas.py` — валидация входных данных;
  - `predictor.py` — инференс модели, риск-категории, факторы;
  - `metrics.py` — счётчики наблюдаемости.
- `utils/` — инфраструктура:
  - `config.py` — загрузка YAML-конфигов и `.env`;
  - `logging.py` — настройка логирования.

## Точки входа

- `python -m src.train` — обучение моделей и сохранение артефактов;
- `python -m src.service` — запуск REST API сервиса.
