# Артефакты проекта

Сюда сохраняются результаты обучения. Файлы создаются командой `python -m src.train` и **не коммитятся** в репозиторий (см. `.gitignore`) — они полностью воспроизводимы.

- `model.joblib` — сериализованный финальный пайплайн (препроцессинг + модель) с метаданными (имя модели, дата обучения, метрики).
- `metrics.json` — сводные метрики финальной модели (CV и тест).
- `model_comparison.csv` — таблица сравнения всех моделей-кандидатов.
- `plots/` — графики: распределение оттока, ROC-кривые, матрица ошибок, значимость признаков.
- `experiments/` — записи запусков обучения (лёгкий трекинг экспериментов, JSON).

Чтобы получить артефакты:

```bash
python -m src.train
```
