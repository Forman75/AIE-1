# Тесты проекта

Набор тестов на `pytest`:

- `test_data.py` — генерация данных, схема признаков, разбиение, воспроизводимость.
- `test_preprocess.py` — препроцессинг: импьютация, one-hot, устойчивость к неизвестным категориям.
- `test_model.py` — реестр моделей, обучение пайплайна, корректность метрик.
- `test_service.py` — эндпоинты REST API (`/health`, `/predict`, `/metrics`), валидация запросов.

## Запуск

```bash
cd project
pytest tests
```

Тесты, требующие обученную модель, автоматически пропускаются, если артефакт `artifacts/model.joblib` ещё не создан (до запуска `python -m src.train`).
